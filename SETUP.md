# Setup on a new machine

This repo is just the Go side. To actually build a patched ISO and test code injection, you need the following set up locally — none of these are committed.

## Required external pieces

### 1. Dolphin emulator
- Install or extract a Dolphin build (this project was developed against Dolphin 2603, x64).
- Note the install path; the dev workflow assumes `Roms/` and `User/` directories live next to `Dolphin.exe`.

### 2. Wind Waker GZLE01 disc image
- A legally obtained Wind Waker (NTSC-U, GameCube) image, CISO or ISO format.
- Place it in Dolphin's `Roms/` folder.

### 3. devkitPPC + Freighter (for compiling the C injection code)
- Install devkitPro with the devkitPPC package: <https://devkitpro.org/wiki/Getting_Started>
- The build expects `C:\devkitpro\devkitPPC` — adjust `build.py` if installed elsewhere.
- `pip install freighter`

### 4. The Freighter project (sibling repo, not in this repo)
- Layout assumed: `~/Desktop/ww-inject/`
  - `src/multiplayer.c` — the injected C code (writes proof-of-life rupees, spawns Player 2)
  - `include/` — game type/symbol headers
  - `build.py` — Freighter build script (sets hook address + inject address)
  - `original.dol` — clean WW main.dol extracted from a known-good disc
  - `patched.dol` — Freighter output (gitignored, regenerated by build.py)
- Run `python build.py` (or `PYTHONUTF8=1 python -X utf8 build.py` on Windows) to regenerate `patched.dol`.
- TODO: this should probably be its own committed repo. For now, copy via USB / sync manually.

### 5. wit (Wiimms ISO Tools)
- Download from <https://wit.wiimm.de/download.html> (Cygwin64 build for Windows).
- Used to convert CISO → plain ISO so we can patch the DOL in place.

### 6. zeldaret/tww decomp (reference only)
- <https://github.com/zeldaret/tww> at `~/Desktop/tww-decomp/`
- Used for symbol lookups (`config/GZLE01/symbols.txt`) and reading game source for hook-target research.

## End-to-end build pipeline

```bash
# 1. Compile the injection (produces ww-inject/patched.dol)
cd ~/Desktop/ww-inject && python build.py

# 2. Convert CISO → plain ISO
wit copy "<roms>/Wind Waker.ciso" "<roms>/WW_Multiplayer_Patched.iso" --iso --trunc --overwrite

# 3. Patch DOL into the ISO and shift the FST (Python snippet — see git history
#    or docs/06-roadmap.md for the exact code; ~20 lines)

# 4. Force Dolphin to re-scan
rm "%APPDATA%/Dolphin Emulator/Cache/gamelist.cache"

# 5. Restart Dolphin and boot WW_Multiplayer_Patched.iso
#    Make sure NO Patches or Gecko Codes are enabled — they fight the DOL.

# 6. Build the Go side
cd ~/projects/ww-multiplayer && go build -o ww.exe .
./ww.exe check    # verify hook + injected code state
./ww.exe debug    # read Link's position for 5 seconds
```
